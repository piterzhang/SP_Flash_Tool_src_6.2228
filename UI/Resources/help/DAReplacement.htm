DA Replacement Feature

 

  For flash tool V6, the DA binary file is along with the load, and default using the DA file in load which configured in the ./download_agent/flash.xml to do flash operations.

  For some other purpose, there are two ways to place the used DA binary file.

 

·  Enter DA Replacement Feature

 

Press Ctrl + Alt + V to enter advanced mode, and the DA Replacement Feature will be shown on GUI just as below.

 

 

 

·  DA Replacement Steps

 

Click the "choose" button, just as below, to select the DA file you want to use.
