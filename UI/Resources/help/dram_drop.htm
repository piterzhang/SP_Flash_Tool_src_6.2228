Drag and Drop Feature

 

  Drag and Drop feature can be used to select flash.xml instead of by clicking the "choose" button.

 

·  Drag and Drop Flash.xml Steps

 

Select the flash.xml, pressing it by mouse-left button and drag it to the partiiton info table(marked as below right region) on the tool, the flash.xml file icon will be shown by a "+" flag, drop it.

After droping the flash.xml, the flash.xml will be selected on Gui.
